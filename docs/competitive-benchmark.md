# 競合ベンチマーク

        ## 比較対象

        - Niagara Launcher: 採用基準は入力負荷、記録粒度、証跡性、ローカル完結性。
- Nova Launcher: 採用基準は入力負荷、記録粒度、証跡性、ローカル完結性。
- Shortcut Maker: 採用基準は入力負荷、記録粒度、証跡性、ローカル完結性。

        ## 採用する基準

        - 入力負荷が低い。
        - 次アクションが明確。
        - ローカルまたは閉域で検証できる。
        - docs と release evidence から状態を追える。

        ## 差別化

        片手ホーム、開発検証アプリ、作業モードを切り替えるAndroidランチャーに絞り、汎用ノートや大型サービスよりも閉域アルファの検証可能性を優先します。
